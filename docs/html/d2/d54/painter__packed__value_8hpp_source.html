74  /*!
75  * \brief
76  * A PainterPackedValue represents a handle to an object that stores
77  * packed state data and tracks if that underlying data is already is
78  * already copied to PainterDraw::m_store.
79  *
80  * If already on a store, then rather than copying the data again, the data
81  * is reused. The object behind the handle is NOT thread safe. In addition
82  * the underlying reference count is not either. Hence any access (even dtor,
83  * copy ctor and equality operator) on a fixed object cannot be done from
84  * multiple threads simutaneously. A fixed PainterPackedValue can be used
85  * by different Painter objects, subject to that the access is done from
86  * same thread.
87  */
88  template<typename T>
